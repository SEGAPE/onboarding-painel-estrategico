# Mapa do Projeto Painel Estratégico

`mapa_projeto_painel_estrategico.xlsx` é a planilha-mapa que mantive à mão como visão única
de todo o Painel Estratégico: o que existe, onde está, qual a fonte e em que status. Foi
construída e atualizada manualmente ao longo do projeto e serviu de bússola para a equipe e
para as demandas ministeriais — é o índice que amarra páginas do painel, tabelas do BigQuery,
métricas e referências.

## Abas

| Aba | Conteúdo |
|---|---|
| Home | Capa e navegação |
| Overview | Visão geral das páginas do painel e seus blocos (89 linhas x 46 colunas) |
| Big Numbers | Indicadores de topo de cada página |
| Dados | Inventário detalhado de métricas e tabelas (1.008 linhas) |
| Experimental | Rascunhos e métricas em estudo |
| Links Rápidos | Atalhos para dashboards, fontes e documentos |
| Tabela Dinâmica | Recortes de análise |
| Google Analytics | Métricas de acesso/uso do painel |
| Data Referência | Datas de referência por indicador |
| Data Referência do Painel | Datas de referência consolidadas (1.244 linhas) |

## Como usar

É material de consulta e planejamento — para localizar rapidamente em qual página do painel
está um indicador, qual tabela do BigQuery o alimenta e qual a data de referência. Ao migrar
o painel para PROD, esta planilha é o checklist do que precisa existir do outro lado.
